AnimeZe is an Android application for streaming anime and reading manga. Designed with custom layouts, it integrates tracking, history, and offline downloads.

## Features

* **Unified Media Platform**: Toggle seamlessly between anime streaming and manga reader.
* **Weekly Airing Schedule**: Live countdowns and airing timelines.
* **Offline Downloads**: High-speed downloader for offline watch and read support.
